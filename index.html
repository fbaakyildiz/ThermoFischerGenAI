Clinical AI decision support

PCT-guided antibiotic stewardship with structured agent review.

BrahmsAI is a local research prototype that combines structured patient inputs, PCT threshold logic, serial biomarker kinetics, prompt-injection controls, and a four-agent report pipeline.

MethodStructured prompts, Pydantic schemas, output validation, safety overrides
DomainPCT-guided antibiotic stewardship for LRTI and sepsis workflows
OutputRecommendation, rationale, kinetic interpretation, warnings, metadata
RuntimeLocal FastAPI app with Gemini, OpenAI, Claude, or OpenRouter keys

Pipeline

Each agent has a narrow role and passes structured evidence forward.

A1

Intake

Validates structured patient data and flags missing or unsafe input content.

A2

Threshold reasoning

Applies LRTI and sepsis PCT bands, gray zones, and recommendation constraints.

A3

Kinetic context

Interprets serial PCT decline, antibiotic day, comorbidities, and treatment-failure risk.

A4

Report

Produces the final stewardship report with warnings, review flags, and next steps.

Run locally

The app is designed for local research use, not public clinical deployment.

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --host 127.0.0.1 --port 8000

Open http://localhost:8000. The browser asks for a model API key for the current local session.

Clinical AI · Antibiotic stewardship

BrahmsAI

A local clinical decision-support prototype for PCT-guided antibiotic stewardship using structured patient input, serial procalcitonin kinetics, and a hardened four-agent LLM pipeline.

Watch demo Open repo Design document

Research only

BrahmsAI is not a medical device and does not replace clinician judgment.

PipelineA1 intake, A2 threshold reasoning, A3 kinetic context, A4 report
RuntimeFastAPI, vanilla JS, local API key gate
ModelsGemini, OpenAI, Claude, OpenRouter

What It Does

PCT interpretation

Applies LRTI and sepsis threshold bands with gray-zone handling.

Kinetic tracking

Calculates serial PCT decline and Day 4 treatment-failure alerts.

Safety flags

Preserves override and clinician-review signals across the pipeline.

Prompt security

Uses input sanitization, hardened prompts, JSON validation, retries, and telemetry.

Agent Pipeline

Patient input
  -> A1 Intake and validation
  -> A2 PCT threshold reasoning
  -> A3 Kinetic and comorbidity context
  -> A4 Final stewardship report

Run Locally

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
